Articulate Fraction Learning Course

An asynchronous instructional module designed for K-6 teachers, built in Articulate Rise 360. Using What Works Clearinghouse (WWC) evidence-based guidelines, this course walks educators through research-backed strategies for teaching fraction comparison, addressing common student misconceptions, and using visual models to build lasting number sense and conceptual understanding.
